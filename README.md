## Project Description

This project was developed as part of the Front-End Coursework 2026 module. The goal of the project was to create a responsive website using HTML and CSS and deploy it using GitHub Pages.

The website includes responsive navigation, Flexbox layouts, CSS Grid layouts, a responsive image gallery, FAQ section, contact links, and a parallax effect. The design adapts to different screen sizes using CSS media queries.
